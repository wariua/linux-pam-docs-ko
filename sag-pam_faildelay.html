6.7. pam_faildelay - 응용별로 실패 시 지연 시간 바꾸기

pam_faildelay.so [ debug ] [ delay=microseconds ]

6.7.1. 설명

pam_faildelay는 응용별로 실패 시 지연 시간을 설정할 수 있는 PAM 모듈이다.

delay를 주지 않으면 pam_faildelay는 /etc/login.defs에서 FAIL_DELAY 값을 가져다 쓴다.

6.7.2. 옵션

debug

syslog로 보내는 디버깅 메시지를 켠다.

delay=N

실패 시 지연 시간을 N 마이크로초로 설정한다.

6.7.3. 제공하는 모듈 종류

auth 모듈 타입만 제공한다.

6.7.4. 반환 값

PAM_IGNORE

지연 시간을 성공적으로 조정했다.

PAM_SYSTEM_ERR

지정한 지연 시간이 유효하지 않다.

6.7.5. 예시

다음 예시는 실패 시 지연 시간을 10초로 설정한다. 

auth  optional  pam_faildelay.so  delay=10000000
