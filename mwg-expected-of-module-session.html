3.4. 세션 관리

올바로 초기화하려면 <security/pam_modules.h>를 포함시키기 전에 PAM_SM_SESSION을 #define 해 두어야 한다. 그러면 정적 모듈을 위한 프로토타입들이 올바로 선언된다.

3.4.1. 세션 관리를 시작하기 위한 서비스 함수

#define PAM_SM_SESSION
#include <security/pam_modules.h>
int pam_sm_open_session(pamh,  
 flags,  
 argc,  
 argv); 
pam_handle_t *pamh;
int flags;
int argc;
const char **argv;
 

3.4.1.1. 설명

pam_sm_open_session 함수는 pam_open_session(3) 인터페이스에 대한 서비스 모듈의 구현이다.

이 함수를 호출해 세션을 개시한다. flags에 유효한 값은 0 또는 다음 값이다.

PAM_SILENT

어떤 메시지도 찍지 않기.

3.4.1.2. 반환 값

PAM_SESSION_ERR

지정한 세션 항목을 생성/제거할 수 없다.

PAM_SUCCESS

세션을 성공적으로 시작했다.

3.4.2. 세션 관리를 종결하기 위한 서비스 함수

#define PAM_SM_SESSION
#include <security/pam_modules.h>
int pam_sm_close_session(pamh,  
 flags,  
 argc,  
 argv); 
pam_handle_t *pamh;
int flags;
int argc;
const char **argv;
 

3.4.2.1. 설명

pam_sm_close_session 함수는 pam_close_session(3) 인터페이스에 대한 서비스 모듈의 구현이다.

PAM_SILENT

어떤 메시지도 찍지 않기.

3.4.2.2. 반환 값
